# 🐍 Classic Snake Game in C++

A fully functional, console-based **Snake Game** developed in C++. This project demonstrates core programming concepts like object-oriented logic, arrays for tail management, game loops, dynamic rendering, and real-time user input handling.

---

## 🚀 Features
* **Real-time Controls:** Smooth movement using `W`, `A`, `S`, `D` keys.
* **Dynamic Tail Mechanism:** The snake grows longer every time it consumes food.
* **Random Fruit Spawning:** Smart algorithmic placement of fruits within the boundary.
* **Score Tracking:** Live score updates instantly as the gameplay progresses.
* **Optimized Rendering:** Uses cross-platform screen clearing logic for a stable console UI.

## 🛠️ Tech Stack & Concepts Used
* **Language:** C++
* **IDE:** Dev-C++ 5.11
* **Core Concepts:** Loops (`for`, `while`), Conditional Logic, `switch-case` structures, Arrays, and Input Buffering (`conio.h`).

## 🎮 How to Play
1. **Run** the application in any C++ compiler (like Dev-C++).
2. Use the following keys to guide the snake:
   * `W` - Move Up
   * `S` - Move Down
   * `A` - Move Left
   * `D` - Move Right
3. Avoid hitting the walls (`#`) or colliding with the snake's own tail, or it's Game Over!
